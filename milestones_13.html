Week 13 April 16, 2014

Question 1

What command would you use to list the contents of a directory on a linux system?

The command ls will list the contents of a directory on a linux system.

Question 2

What command would you use to read the "manual page" for a specific command?

You use the man command in combination with the specific command you are interested in. For instance, if you wanted the manual for the ls command, you would enter man ls .
