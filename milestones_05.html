Question 1:

Briefly describe each dataset based upon the information from the web site.

Dataset #1

New Mexico Known Mineral Deposit Areas

This vector dataset contains all Known Mineral Deposit Areas in the State of New Mexico. It was digitized from a 1:500,000 scale map of the state. It is intended for use as a general reference to the extent and boundaries of Known Mineral Deposits in NM.

RGIS Metadata

Dataset #2

Digital Geologic Map of New Mexico - Volcanic Vents

This shapefile shows the volcanic vents for the State of New Mexico that are used on the Digital Geologic Map for New Mexico. The file was created in GSMAP at Socorro, NM by Orin Anderson and Glen Jones and published as the Geologic Map of New Mexico 1:500,000 in 1997.
